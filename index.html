Quick Primer

初めてでも分かる、米プレラの見方

プレラとは？

プレミア公開が近い時刻に連続する現象、またはそれを目的としたイベントのことです。普通の動画一覧ではなく、時間に合わせて見る催しです。

プレミア公開とは？

事前に用意した動画を指定時刻にみんなで同時視聴する YouTube の形式です。各動画に専用ページがあり、「通知を受け取る」やチャット参加ができます。

Glossary

用語と見方

プレミアラッシュ / プレラ

プレミア公開が近接して連続する現象、またはそれを目的としたイベントのことです。

プレミア公開

指定時刻に動画が始まり、専用ページで同時視聴とチャットができる YouTube の公開形式です。

親

このページでの「親」はプレミア公開が始まってからの待機時間です。動画申し込みでは 1〜5 の数字で選びます。

被り

見たい動画どうしの時間が重なることです。被りそうなときはタブを分けると追いやすくなります。

枠

このページでは動画の系譜や模倣元ごとの分類を枠として表示しています。タイムテーブル上では同じ枠の並びも追いやすくなります。

刻む

前後の予定を見ながら細かく時刻を調整して入れることです。短い動画ほど刻みやすいです。

ゲリラ

事前告知が少ないまま飛び込むように参加・公開することです。補足欄に一言あると見に来る人に優しいです。

公式予定 / 参加動画 / 審査待ち

公式予定は運営側の掲載、参加動画は応募作品、審査待ちはまだ一般タイムテーブルに出ていない確認中の状態です。

枠の見方

このページでは界隈で使われる模倣元の系譜に合わせて、DSC枠 / 逆再生枠 / .枠 / ..枠 / ...枠 / 表/裏枠 / アブジェ枠 / 教育枠 / 名の星枠 / エヌ枠 / K²枠 / オリジナル枠 で整理しています。

申し込みの流れ
